# Minimal Store

A sophisticated, minimalistic e-commerce interface designed for a luxury fashion brand featuring the "Noir" collection.

## Features
- **Modern UI/UX**: Clean, dark-themed design built with Tailwind CSS.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Interactive Shopping Cart**: A sleek side-sliding cart panel for a seamless shopping experience.
- **Product Gallery**: Dynamic grayscale-to-color hover effects on product images.
- **Custom Cursor**: A personalized cursor to enhance the premium feel of the site.

## Technologies Used
- HTML5
- [Tailwind CSS](https://tailwindcss.com/) (CDN)
- JavaScript (Vanilla)
- Google Fonts (Inter)

## Getting Started
1. Clone or download the repository.
2. Open `index.html` in any modern web browser to view the store.

## Project Structure
- `index.html`: The main landing page including the hero section and product grid.
- `about-us.html`: The brand story page.
- `assets/`: Contains custom `style.css` and `script.js` for site functionality.
